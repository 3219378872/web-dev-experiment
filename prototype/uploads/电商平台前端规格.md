# 电商平台 — 前端规格文档

> 从《Web开发技术》实验指导书中提取，按模块与优先级梳理。

---

## 1 技术栈

| 层面 | 技术选型 |
|------|---------|
| 框架 | Vue 3 |

| 状态管理 | Pinia |

## 2 客户端（C 端）

### 2.1 商品模块 ★ 基本要求

| 页面 / 功能 | 详细说明 |
|------------|---------|
| 分类浏览页 | 按分类筛选商品列表 |
| 商品详情页 | 商品图片、名称、价格、库存、参数、详情介绍 |
| 搜索结果页 | 精准搜索、模糊搜索；排序维度：价格 / 销量 / 新品 |
| 收藏功能 | 收藏 / 取消收藏、收藏列表查看 |
| 商品评价 | 查看评价列表、提交评价、上传评价图片 |

### 2.2 购物车模块 ★ 基本要求

| 页面 / 功能 | 详细说明 |
|------------|---------|
| 添加购物车 | 选择规格、数量后加入 |
| 购物车列表 | 展示商品图片、名称、单价、数量、小计 |
| 购物车操作 | 修改数量、删除商品、勾选商品、全选 / 反选 |
| 结算跳转 | 跳转至订单确认页 |

### 2.3 订单模块 ★ 基本要求

| 页面 / 功能 | 详细说明 |
|------------|---------|
| 订单确认页 | 选择收货地址、商品清单、总价计算 |
| 订单提交 | 生成订单号、选择支付方式 |
| 订单支付 | 模拟支付 / 对接支付接口 |
| 我的订单 | 按状态分 Tab：全部 / 待支付 / 待发货 / 待收货 / 已完成 / 已取消 |
| 订单操作 | 取消订单、确认收货、申请退款、查看物流 |

### 2.4 用户认证模块 ☆ 进阶要求

| 页面 / 功能 | 详细说明 |
|------------|---------|
| 用户注册页 | 邮箱注册、验证码验证 |
| 用户登录页 | 账号密码登录、记住密码、退出登录 |
| 找回密码页 | 邮箱验证、重置密码 |
| 个人信息编辑页 | 头像、昵称、邮箱、密码修改 |

### 2.5 首页模块 ☆ 进阶要求

| 页面 / 功能 | 详细说明 |
|------------|---------|
| 轮播广告 | 广告图展示 |

| 商品展示区 | 热门商品、新品推荐、促销商品 |
| 搜索框 | 关键词搜索、热门搜索词 |

### 2.6 收货地址模块 ☆ 进阶要求

| 页面 / 功能 | 详细说明 |
|------------|---------|
| 新增地址 | 收货人、手机号、省市区、详细地址 |

### 2.7 其他进阶功能

## 4 界面与交互规格

| 维度 | 要求 |
|------|------|
| 视觉风格 | 简洁美观、布局统一 |
| C 端侧重 | 突出商品展示与购买流程 |
| B 端侧重 | 提升管理效率 |
| 操作反馈 | 所有操作需有明确成功 / 失败提示 |
| 质量底线 | 杜绝布局错乱、功能失效 |
| 响应式布局 | 适配手机 / 电脑（进阶） |
| 图片上传 | 商品图、评价图、轮播图等（进阶） |
| 数据分页 | 列表类页面需分页（进阶） |
