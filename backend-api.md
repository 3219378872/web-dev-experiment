# 后端接口缺失/待完善记录

> 本文件记录前端对齐高保真原型过程中发现的后端接口缺失或不完备项。
> 技术栈：Vue3 + Element Plus

## 客户端 hmall-web

### 1. 首页 (Home)
- [ ] **轮播图接口** —— 需要获取首页轮播图列表（图片URL、跳转链接）
- [ ] **限时秒杀接口** —— 需要获取当前秒杀活动商品列表（含倒计时、库存百分比）
- [ ] **分类金刚区接口** —— 需要获取热门分类图标/名称列表
- [ ] **广告位接口** —— 需要获取双广告位配置数据
- [ ] **用户简要信息接口** —— 首页侧边栏需要展示用户昵称、会员等级、积分
- [ ] **优惠券列表接口（首页）** —— 首页侧边栏需要展示新人专享券列表

### 2. 商品详情 (ItemDetail)
- [ ] **商品评价接口** —— 需要获取商品评价列表（含评分、评论内容、图片）
- [ ] **商品规格接口** —— 需要获取SKU规格选择数据
- [ ] **相关推荐接口** —— 需要获取相关商品推荐

### 3. 购物车 (Cart)
- [ ] 当前接口基本满足，但需确认是否支持：
  - 勾选/全选/反选状态同步
  - 数量修改实时价格计算

### 4. 订单确认 (OrderConfirm)
- [ ] **收货地址列表接口** —— 需要获取用户地址列表并支持选择默认地址
- [ ] **优惠券选择接口** —— 需要获取当前可用优惠券列表
- [ ] **运费计算接口** —— 需要根据地址和商品计算运费

### 5. 订单列表 (OrderList)
- [ ] **订单状态筛选接口** —— 需支持按状态Tab筛选（全部/待付款/待发货/待收货/已完成/已取消）
- [ ] **订单取消接口** —— 用户主动取消订单
- [ ] **确认收货接口** —— 用户确认收货
- [ ] **申请退款接口** —— 用户申请退款/售后

### 6. 订单详情 (OrderDetail) —— 新页面
- [ ] **订单详情接口** —— 获取订单完整信息（商品、地址、支付、物流时间轴）
- [ ] **物流跟踪接口** —— 获取物流时间轴数据

### 7. 收藏夹 (FavoriteList)
- [ ] 当前接口基本满足，但需确认是否支持批量取消收藏

### 8. 收货地址 (AddressList)
- [ ] **地址CRUD接口** —— 需确认是否完整支持增删改查、设置默认地址

### 9. 个人信息 (Profile)
- [ ] **用户信息更新接口** —— 需支持头像上传、昵称、邮箱、密码修改
- [ ] **头像上传接口** —— 需要独立的头像上传接口或复用 file-service

### 10. 优惠券 (Coupons) —— 新页面
- [ ] **优惠券列表接口** —— 获取用户已领取的优惠券
- [ ] **领取优惠券接口** —— 用户主动领取优惠券
- [ ] **可用优惠券查询接口** —— 下单时查询可用优惠券

### 11. 限时秒杀 (FlashSale) —— 新页面
- [ ] **秒杀活动接口** —— 获取当前秒杀活动列表
- [ ] **秒杀倒计时接口** —— 活动开始/结束时间

### 12. 在线客服 (Service) —— 新页面
- [ ] **客服消息接口** —— 发送/接收客服消息（WebSocket或轮询）
- [ ] **常见问题接口** —— 获取常见问题列表

### 13. 系统公告 (Notifications)
- [ ] **公告列表接口** —— 获取系统公告列表（分页）
- [ ] **公告详情接口** —— 获取单条公告详情

### 14. 用户反馈 (Feedback)
- [ ] **反馈提交接口** —— 需确认字段是否足够（类型、内容、联系方式、图片）
